## Технологии

**Текущий стек:**
- Python 3.x
- matplotlib (для визуализации)

**Рекомендации для MVP:**
- Python для логики игры
- pygame или pyglet для визуализации (или оставить matplotlib для начала)
- JSON для данных

## Структура проекта

```
awaking/
├── data/              # JSON данные (уже есть)
├── src/
│   ├── core/         # Ядро игры
│   │   ├── game.py   # Главный игровой цикл
│   │   ├── time.py   # Система времени
│   │   └── world.py  # Мир и карта
│   ├── entities/     # Сущности
│   │   ├── unit.py   # Юниты/отряды
│   │   ├── node.py   # Узлы
│   │   └── kingdom.py # Королевства
│   ├── mechanics/    # Механики
│   │   ├── economy.py # Экономика
│   │   ├── combat.py  # Бой
│   │   └── relations.py # Отношения
│   └── utils/        # Утилиты
│       ├── pathfinding.py # Поиск пути
│       └── loader.py  # Загрузка данных
├── tests/            # Тесты
└── requirements.txt  # Зависимости
```
