# Data

**Статическая карта и артефакты генерации карты.**

## Назначение

`data/` содержит только:
- Статическую карту (JSON файлы)
- Артефакты генерации карты
- Координаты и метаданные карты

## Правило

**`data/` — только статическая карта и артефакты генерации карты.**

❌ **Не кладём сюда:**
- Королевства (→ `lore/tables/kingdoms.*`)
- Глоссарий (→ `lore/glossary/`)
- Механики (→ `game_data/` или `design/`)
- Лор (→ `lore/`)

## Структура

```
data/
├── maps/                    # Данные карты
│   ├── map_coordinates.json # Координаты для генерации
│   ├── simple_map.json      # Простая тестовая карта
│   ├── world_v1.json        # Основная статическая карта
│   └── README.md
├── generated_map.json       # Сгенерированная карта (gitignored)
└── README.md                # Этот файл
```
