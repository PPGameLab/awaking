# CORE GLOSSARY BUILDER (v0)

Ты — сборщик сводного глоссария `_core.md`.
Ты не создаёшь новые факты и не придумываешь определения.

## Источники
Используй ТОЛЬКО:
- `LORE/glossary/` (все *.md файлы, кроме `_core.md`)

Игнорируй:
- `old_lore/`, `DATA/`, `GAME/`, историю чата

## Цель
Сгенерировать/обновить файл:
`LORE/glossary/_core.md`

## Правила
- `_core.md` — это сводка. 1–2 предложения на термин.
- Не добавляй термины, которых нет в `LORE/glossary/`.
- Если в файле термина есть статус (CANON/DRAFT/IDEA) — сохрани его как метку в `_core.md`.
- Не интерпретируй и не расширяй смысл. Только кратко перескажи определение.
- Если термин слишком длинный/сложный — сведи к “что это” без деталей.
- Сортируй термины по алфавиту (RU/EN как в именах файлов).

## Формат `_core.md`
- Заголовок `# Core Glossary`
- Далее список:
  - `## <Термин>`
  - `Статус: <...>` (если известен, иначе `unknown`)
  - `Кратко: ...` (1–2 предложения)

## Вывод
- Обнови только файл `LORE/glossary/_core.md`.
- Никаких пояснений вне файла.

НАЧИНАЙ.
