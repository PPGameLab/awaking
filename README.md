# Awaking — фэнтези-стратегия

Игра в жанре гибрид стратегии и тактики, ориентированная на Battle Brothers и Mount & Blade.

## Текущее состояние

**Статус:** в разработке MVP

**Сейчас:** лор и данные (LORE/, DATA/) приведены в порядок; **код и всё, что связано со старой картой, удалены.** Дальнейшая разработка — с нуля, без опоры на старую реализацию.

## Структура проекта

```
awaking/
├── LORE.md              # Вход в лор (см. также LORE/)
├── DATA.md              # Вход в данные (см. также DATA/)
├── GAME.md              # Вход в код/игру (код будет добавляться заново)
├── WORKFLOW.md          # Рабочий процесс
├── WORLD_SNAPSHOT.md    # Слепок мира для AI (автогенерация)
│
├── LORE/                # Лор, глоссарий, канон
│   └── glossary/        # Термины (1 файл = 1 термин)
│       ├── _core.md     # Сводка (автогенерация: BUILD_CORE_GLOSSARY)
│       ├── characteristics/   # Характеристики (сила, ловкость, …)
│       ├── map/         # Термины карты (позиция, движение, POI, …)
│       └── meta/        # Мета-термины (актор, отряд, контролёр, …)
│
├── DATA/                # Структурированные данные мира
│   └── characteristics.json   # Словарь характеристик (id, name, lore)
│
├── PROMPTS/             # Промпты для Cursor (сборка, валидация)
│   ├── INDEX.md         # Список команд (CMD: BUILD_SNAPSHOT и др.)
│   └── …
│
├── tools/               # Утилиты (list_prompts и др.)
├── uwuking.map         # Карта из Azgaar (референс)
└── requirements.txt    # Зависимости (на момент очистки — без запускаемого кода)
```

**Истина:** лор — в `LORE/glossary/`; структурированные сущности — в `DATA/`. Подробнее: `LORE.md`, `DATA.md`.

## Команды Cursor (PROMPTS)

В чате можно вызывать: `CMD: BUILD_SNAPSHOT`, `CMD: BUILD_CORE_GLOSSARY`, `CMD: NEW_GLOSSARY_TERM`, `CMD: VALIDATE_LORE_DATA` и др. Полный список — в **`PROMPTS/INDEX.md`**.

## Основные механики (по лору)

- **Карта:** узловая система, статическая карта (термины в `LORE/glossary/map/`).
- **Юниты:** отряды, контролёр (актор / персонаж игрока).
- **Время:** непрерывное с тиками, календарь.
- **Характеристики:** 8 характеристик (см. `DATA/characteristics.json`, `LORE/glossary/characteristics/`).

Реализация кода и карты — заново.
