# Система Характеристик

**Status:** CANON-DRAFT

## Фундаментальные Характеристики

В мире существует 8 основных характеристик, которые определяют способности каждого юнита (персонажа):

1. **Сила** (Strength) - физическая мощь, способность наносить урон, поднимать тяжести
2. **Ловкость** (Dexterity) - скорость, точность, координация движений
3. **Выносливость** (Endurance) - стойкость, способность переносить нагрузки и повреждения
4. **Дух** (Spirit) - внутренняя сила, воля, связь с духовным
5. **Жадность** (Greed) - стремление к накоплению, материальным благам, власти
6. **Удача** (Luck) - случайность, везение, влияние судьбы
7. **Интеллект** (Intelligence) - логика, анализ, технические знания
8. **Мудрость** (Wisdom) - понимание, опыт, интуиция, связь с природой

## Связь с Королевствами

Каждое королевство неявно отражает одну из характеристик:

**Status:** CANON
- Королевство Креста → **Жадность** (касты, богатство, алтари, усыпальницы)
- Королевство Инженеров → **Интеллект** (наука, технологии, логика)
- Королевство Магии Тела → **Выносливость** (физическое совершенство, стойкость)
- Вулканическое Королевство → **Сила** (кузнечное дело, горная промышленность)
- Потустороннее Королевство → **Удача** (магия удачи, сделки)
- Край Природных Защитников → **Мудрость** (единение с природой, понимание)
- Королевство Стойкости и Духа → **Дух** (внутренняя сила, медитация)
- Северо-Западный край → **Ловкость** (охота, рыболовство, выживание)

## Магия в Мире

**Status:** CANON

Магия доступна грубо говоря каждому королевству, но со многими условиями и ограничениями.

**Важно:** Система магии описана ниже касается **только игрока**. У юнитов нанимаемых собственные шаблонные заклинания.

### Магические Школы

**Status:** CANON

8 основных и мажорных школ магии:

| Школа | Характеристика | Основная роль |
|-------|----------------|---------------|
| **Огонь / Земля** | Сила | Прямой урон, разрушение |
| **Тело** | Выносливость | Броня, регенерация |
| **Мехи** | Интеллект | Перезарядка, механизация |
| **Природа** | Мудрость | Яды, поддержка, контроль зон |
| **Вода / Ветер** | Ловкость | Скорость и уклонение |
| **Дух** | Дух | Защита, неуязвимость |
| **Демоны** | Удача | Критический урон, хаос |
| **Некромантия** | Жадность | Души, масштабирование |

### Общая Магическая Система

**Status:** CANON

- **Система общая** для всего мира. Королевства — это экономическая и социальная зона, а не магическая изоляция
- **Законы магии общие** на весь мир — можно использовать магию другого королевства
- В каждом королевстве **лучше всего условия для продвижения в конкретной школе** (соответствующей характеристике королевства)
- **Все характеристики влияют на магию** по-разному, в зависимости от школы

### Система Заклинаний

**Status:** CANON

- У каждой школы **2 активных заклинания** и **1 пассивка** (всего 16 активных + 8 пассивных = 24 заклинания)
- Заклинания имеют **10 уровней**
- На 10-м уровне возможны **эволюции**:
  1. **Эволюция с соседней школой** → комбинированное заклинание (заменяет исходные)
  2. **Эволюция с пассивкой школы** → внутренняя форма (сохраняет школу)

**Пример развития заклинания (Огненный шар):**
- Уровень 1: +10% урона
- Уровень 2: +20% радиус
- Уровень 3: +20% урона
- Уровень 4: +1 снаряд
- Уровень 5: Оставляет огненную землю
- ...
- Уровень 10: Возможность эволюции

**Комбинации:** Фиксированы. Комбинированные спеллы заменяют исходные. Внутренние (через пассивку) сохраняют школу. Ультимативные эволюции появятся позже.

### Пассивные Способности Школ

**Status:** CANON

| Школа | Характеристика | Эффект |
|-------|----------------|--------|
| Огонь / Земля | Сила | +% урона всем заклинаниям |
| Тело | Выносливость | +HP и регенерация |
| Мехи | Интеллект | -% перезарядки |
| Природа | Мудрость | +% размера заклинаний |
| Вода / Ветер | Ловкость | +% скорости движения и снарядов |
| Дух | Дух | +% сопротивления, уклонение |
| Демоны | Удача | +% критического урона |
| Некромантия | Жадность | Каждые N убийств = +1 характеристика |

- 10 уровней, каждые 3–5 уровней — мини-перк
- Усиливаются через Цитадель и архетипы
- Отображаются в отдельной панели рядом с заклинаниями

### Прогрессия через Магию

**Status:** CANON

- **Изучение заклинаний школы** повышает связанную характеристику
- **Вторичные параметры** выводятся из комбинаций характеристик
- **Софт-ресет:** характеристики сбрасываются ради перманентного бонуса

### Цена Магии

**Status:** CANON-DRAFT

- **Мана** — основной ресурс
- **Кулдаун** — время восстановления
- **Условия для конкретных случаев** — специфические требования для определённых заклинаний

**Данные:** Структурированные данные в `data/magic_system.json`

## Повышение Характеристик

**Status:** CANON-DRAFT

- **Обычные юниты** (отряды, солдаты, наёмники): статичные характеристики, при достижении уровня могут улучшиться по ветке улучшений, с эволюцией и развитием
- **Герои** (особые персонажи): качаются по другой системе, непрерывно повышая характеристики

**Ограничения:**
- Минимальное значение: **1** (отрицательных нет)
- Максимальное значение: в разработке

**Вторичные характеристики:**
- Вторичные характеристики будут в виде навыков — определённых, более узконаправленных способностей
