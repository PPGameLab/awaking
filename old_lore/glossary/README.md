# Глоссарий

Каждый термин — отдельный файл.
Никакие игровые механики здесь не описываются.

## Правила

**Глоссарий = только определения.**

❌ **Запрещено:**
- "story hooks"
- "open questions" внутри записи
- списки "relevantKingdoms" (связи → `lore/tables/kingdom_term_links.csv`)

✅ **Разрешено:**
- Определение термина
- Краткое описание
- Контекст использования

## Статусы

- **CANON** — зафиксировано
- **DRAFT** — в работе
- **IDEA** — гипотеза

⚠ **CANON-DRAFT запрещён.**  
⚠ **Статус только в метаданных файла, не внутри текста.**

## Структура

- Термин → `lore/glossary/<term>.md` (1 файл = 1 термин)
- Связи термин↔королевство → `lore/tables/kingdom_term_links.csv`
- Вопросы → `lore/meta/open_questions.md`

Формат термина см. `TEMPLATE.md`
