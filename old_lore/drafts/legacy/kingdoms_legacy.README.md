# kingdoms_legacy.json

⚠ **LEGACY DATA DUMP**

Файл содержит смешанные данные (CANON-DRAFT/PROPOSAL/openQuestions).
Не используется как источник истины.

Актуальные данные должны быть перенесены в:
- `lore/tables/` (структурные поля)
- `lore/canon/` (зафиксированный нарратив)
- `lore/glossary/` (термины)
