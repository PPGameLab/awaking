# LORE

Здесь хранится человекочитаемый лор, глоссарий, канон, черновики, рассуждения.

## Структура

| Путь | Назначение |
|------|------------|
| `LORE/glossary/` | Глоссарий терминов мира (1 файл = 1 термин). **Истина здесь.** |
| `LORE/glossary/_core.md` | Автогенерируемая сводка из файлов выше. Не источник истины; при расхождении верен файл термина. |

## Правила

- Истина — в файлах; термина нет в glossary — его нет в мире.
- Статусы: **CANON** / **DRAFT** / **IDEA** (CANON-DRAFT не используем).
- Глоссарий — только определения; вопросы и связи — в отдельные файлы/таблицы.
