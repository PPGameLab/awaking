# Assets для карты

Эта папка содержит графические ресурсы для карты.

## Структура

```
assets/maps/
├── graphics/          # Графические ресурсы
│   ├── node_icons/   # Иконки узлов (будущее)
│   ├── terrain/      # Текстуры ландшафта (будущее)
│   └── roads/        # Текстуры дорог (будущее)
└── README.md
```

## Использование

Графика будет использоваться для:
- Иконок узлов (разные для Capital, City, Outpost и т.д.)
- Текстур ландшафта
- Текстур дорог
- Других визуальных элементов
